GraphicalModelCreator

A frame work for creating interactive probabilistic graphical models for visualizing the theory of communication using D3.js and ES6

  • Documentation - Can be found in GraphicModelCreator/Documentation/out.
  • Examples and Tutorial - Can be found in GraphicModelCreator/Tutorial.
  • Comptability - ES6 is only supported in Chrome at the moment, use Babel to transpile the code to ES5 to achieve full compatability in Firefox and Safari.
  • _Documentation - Generated using jsDoc.
